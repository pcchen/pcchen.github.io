    量子計算模擬與實作¶

    Assignments¶

    1. Implement Deutsch-Jozsa algorithm (without noise)
    2. Implement T1, T2, T2* experiment (with noise)
    3. Implement Deutsch-Jozsa algorithm (with noise)
    4. Run HW(1) and HW(2) on real device
    5. Implement Grover search algorithm (without noise). You should first implement the quantum oracle \(U_f(|x\rangle_n |y\rangle_1)=|x\rangle_n|y\oplus f(x)\rangle_1\). First try 3-bit integer x, then try 5-bit integer x. (without noise)
    6. Implement the VQE in these tutorials
    7. https://qiskit.org/ecosystem/algorithms/tutorials/01_algorithms_introduction.html
    8. https://qiskit.org/ecosystem/algorithms/tutorials/02_vqe_advanced_options.html
    9. https://qiskit.org/ecosystem/algorithms/tutorials/03_vqe_simulation_with_noise.html
    10. modify to use fakebackend with noise, and try on real device.
    11. Implement following with statevector, fakebackend with noise, real device
    12. Quantum Real Time Evolution using Trotterization. Try different order.
    13. Variational Quantum Time Evolution
    14. Run a 8-site real time evolution with two algorithms and compare the results.
